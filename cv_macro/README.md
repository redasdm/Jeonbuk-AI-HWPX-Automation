# AntiGravity CV 자동 수강 매크로

브라우저 확장 프로그램 없이 **OpenCV 화면 인식**으로 자동으로 강의를 수강합니다.

## 작동 원리

| 상태 | 감지 방법 | 행동 |
|------|-----------|------|
| 🟢 재생중 배지 | HSV 초록색 색상 감지 | 대기 |
| 재생중 → 사라짐 | 초록색 없어짐 | 다음 항목 탐색 시작 |
| 📋 학습전 배지 발견 | 템플릿 이미지 매칭 | 배지 왼쪽 텍스트 클릭 |
| 모든 항목 학습완료 | 학습전 없음 | 다음 차시 버튼 클릭 |

## 처음 설치 (1회만)

1. **`run.bat` 더블클릭**
2. `[1] Capture template images` 선택
3. 강의 팝업창을 크롬에서 열어 두기
4. 안내에 따라 `학습전` 배지와 `다음 차시` 버튼을 드래그로 캡처

## 매크로 실행

1. **`run.bat` 더블클릭**
2. `[2] Run CV macro` 선택
3. 강의가 자동으로 진행됨
4. 종료하려면 이 창에서 `Ctrl+C`

## 파일 구조

```
cv_macro/
├── cv_macro.py           ← 메인 매크로 스크립트
├── capture_templates.py  ← 템플릿 이미지 캡처 도구
├── run.bat               ← 설치 + 실행 배치파일
├── requirements.txt      ← Python 패키지 목록
└── templates/            ← 캡처된 템플릿 이미지 폴더
    ├── badge_before.png       (캡처 후 자동 생성)
    └── btn_next_chapter.png   (캡처 후 자동 생성)
```

## 설정 변경 (`cv_macro.py` 상단)

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `SCAN_INTERVAL` | `2.0` | 화면 감시 주기 (초) |
| `MATCH_THRESHOLD` | `0.75` | 템플릿 매칭 민감도 (낮추면 더 잘 찾음) |
| `CLICK_OFFSET_X` | `-80` | 학습전 배지 기준 왼쪽 클릭 거리 (px) |
| `SIDEBAR_REGION` | `None` | 사이드바 스캔 영역 제한 `(x, y, w, h)` |

## 문제 해결

- **템플릿을 못 찾는 경우**: `MATCH_THRESHOLD`를 `0.65`로 낮춰보세요
- **클릭 위치가 빗나가는 경우**: `CLICK_OFFSET_X` 값을 조정하세요 (더 음수로)
- **속도가 너무 빠른 경우**: `SCAN_INTERVAL`을 `3.0`~`5.0`으로 늘려보세요
