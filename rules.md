# 프로젝트 규칙 (Project Rules)

이 프로젝트에서 코드를 작성하거나 수정할 때는 다음 규칙을 반드시 준수해 주세요.

## 1. 언어 규칙 (Language Rules)
- **모든 설명 자료는 반드시 한글로 작성해야 합니다.**
- README, 기획 문서, 가이드 문서, 에러 메시지(사용자 노출용) 등 프로젝트 내의 모든 부가 설명에는 한국어를 사용합니다.

## 2. 코드 주석 (Code Comments)
- **중요한 함수, 클래스, 핵심 로직 및 기능에는 반드시 한글로 주석을 달아주세요.**
- 주석은 코드의 동작 방식(How)보다 의도와 목적(Why)을 명확하게 설명해야 합니다.
- **예시:**
  ```python
  # 사용자 인증 토큰을 검증하고, 유효한 경우 사용자 정보를 반환하는 클래스
  class UserAuthenticator:
      def check_token(self, token):
          # 만료된 토큰인지 확인 (보안 강화 목적)
          ...
  ```
